# 🚀 Proyecto React + MongoDB Atlas

Este es un proyecto web desarrollado con **React** y un backend conectado a **MongoDB Atlas**.

---

## 📌 Descripción

La aplicación cuenta con un sistema de autenticación que permite:

- 🔐 Inicio de sesión de usuarios
- ☁️ Conexión con base de datos MongoDB Atlas
- 📊 Acceso a un dashboard después del login
- 🚪 Cierre de sesión seguro

El proyecto está diseñado con una interfaz moderna y funcional, enfocada en la experiencia del usuario.
